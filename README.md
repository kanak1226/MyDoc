# 🏥 MyDoc - Doctor Appointment Booking System
    
MyDoc is a full-stack Doctor Appointment Booking System developed using the MERN Stack. It allows patients to book appointments with doctors, while doctors and administrators can efficiently manage appointments, profiles, and system data.

---

## 🚀 Features

### 👤 User
- User Registration & Login
- Browse Doctors
- Book Appointments
- View Appointment History
- Update Profile
- Cancel Appointments

### 👨‍⚕️ Doctor
- Doctor Login
- Dashboard
- Manage Profile
- View Appointments
- Update Availability

### 🛠️ Admin
- Admin Login
- Dashboard
- Add New Doctors
- Manage Doctors
- Manage Appointments
- View System Statistics

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/kanak1226/MyDoc.git
```

### Navigate to the Project Directory

```bash
cd MyDoc
```

### Install Dependencies

```bash
npm install
```

### Start the Development Server

```bash
npm run dev
```
